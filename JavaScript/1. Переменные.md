## Что такое переменная?

Переменная — это **именованная область памяти**, где хранится значение. В JavaScript переменные **динамически типизированы** — тип определяется автоматически по значению.

Представьте переменную как коробку с ярлыком, но размер коробки меняется автоматически под содержимое.

## Объявление переменных

**ES6+ (современный способ):**

javascript

`let age = 25;           // изменяемая переменная const name = "Алексей"; // константа (не изменяется)`

**Старый способ (ES5):**

javascript

`var salary = 75000.5;   // функциональная область видимости`

**Рекомендация:** используйте только `let` и `const`!

## Типы данных

## Примитивные типы (7 штук)

|Тип|Описание|Пример|
|---|---|---|
|`number`|Число (целое/дробное)|`42`, `3.14`|
|`bigint`|Большое целое число|`1234567890123456789n`|
|`string`|Строка|`"текст"`, `'текст'`|
|`boolean`|Логическое (true/false)|`true`, `false`|
|`null`|"Ничего" (специальное значение)|`null`|
|`undefined`|Не определено|`let x;`|
|`symbol`|Уникальный идентификатор|`Symbol('id')`|

## Ссылочные типы

javascript

`let person = { name: "Иван", age: 30 };  // объект let colors = ["красный", "синий"];       // массив let func = function() { return "привет"; }; // функция`

## Полный пример программы

javascript

`// Объявление переменных let age = 30; let height = 1.75; const name = "Иван"; const isStudent = true; let bloodType = 'A'; console.log("Имя:", name); console.log("Возраст:", age, "лет"); console.log("Рост:", height, "м"); console.log("Студент:", isStudent); console.log("Группа крови:", bloodType); // Арифметические операции let nextAge = age + 1; let weight = 75.5; let bmi = weight / (height * height); console.log("BMI:", bmi); console.log("Возраст через год:", nextAge);`

**Вывод в консоль:**

text

`Имя: Иван Возраст: 30 лет Рост: 1.75 м Студент: true Группа крови: A BMI: 24.48979591836735 Возраст через год: 31`

## Разница let, const и var

javascript

`let changeable = 10; changeable = 20;  // ✅ OK const constant = 100; // constant = 200; // ❌ TypeError! var oldStyle = 5; // Избегаем var`

## Именование переменных

## Правила:

- Начинается с буквы, `_`, `$` или цифры **нет**
    
- Может содержать буквы, цифры, `_`, `$`
    
- **Регистр важен!** `age` ≠ `Age`
    

## Конвенции (camelCase):

javascript

`// ✅ Хорошо let userAge; let firstName; let monthlySalary; let isLoggedIn; // ❌ Плохо let age1; let name_of_user; let SALARY;`

## Автоматическое приведение типов

javascript

`let num = 5; let text = "10"; console.log(num + text);     // "510" (склеивание строк) console.log(num + +text);    // 15 (число + число) console.log(num * text);     // 50 (авто-приведение к числу)`

## Области видимости

javascript

`let globalVar = "глобальная"; if (true) {     let blockVar = "блочная";  // видна только внутри блока    console.log(globalVar);    // ✅ работает    console.log(blockVar); } console.log(globalVar);    // ✅ работает // console.log(blockVar);  // ❌ ReferenceError! for (let i = 0; i < 3; i++) {     console.log("i =", i); } // console.log(i);  // ❌ ReferenceError!`

## Константы и математические значения

javascript

`const PI = 3.14159; const MAX_USERS = 100; console.log("PI =", PI); console.log("Максимум пользователей:", MAX_USERS);`

## Проверка типа данных

javascript

`let age = 25; let name = "Иван"; console.log(typeof age);   // "number" console.log(typeof name);  // "string" console.log(typeof null);  // "object" (историческая ошибка)`
