## Что такое переменная?

Переменная — это **именованная область памяти**, где хранится значение определённого типа. Представьте переменную как коробку с ярлыком: ярлык — это имя переменной, а внутри коробки лежит данные нужного размера и типа.

В Java **строгая типизация**: каждая переменная должна иметь явно указанный тип данных перед использованием.

## Объявление и инициализация

**Синтаксис объявления:**

java

`тип имя_переменной = значение;  // с инициализацией // или тип имя_переменной;           // только объявление`

**Пример:**

java

`int age = 25;           // целое число double salary = 75000.5; // число с плавающей точкой String name = "Алексей"; // строка boolean isActive = true; // логическое значение char grade = 'A';        // один символ`

## Основные типы данных

## Примитивные типы (8 штук)

| Тип       | Размер  | Диапазон значений          | Пример объявления      |
| --------- | ------- | -------------------------- | ---------------------- |
| `byte`    | 1 байт  | -128 до 127                | `byte b = 100;`        |
| `short`   | 2 байта | -32,768 до 32,767          | `short s = 10000;`     |
| `int`     | 4 байта | -2^31 до 2^31-1 (~2 млрд)  | `int i = 123456;`      |
| `long`    | 8 байт  | -2^63 до 2^63-1            | `long l = 123456L;`    |
| `float`   | 4 байта | ~6-7 значащих цифр         | `float f = 3.14f;`     |
| `double`  | 8 байт  | ~15 значащих цифр          | `double d = 3.14159;`  |
| `char`    | 2 байта | Unicode символы (0-65,535) | `char c = 'A';`        |
| `boolean` | 1 бит   | `true` или `false`         | `boolean flag = true;` |

## Полный пример программы

java

`public class VariablesExample {     public static void main(String[] args) {        // Объявление и инициализация        int age = 30;        double height = 1.75;        String name = "Иван";        boolean isStudent = true;        char bloodType = 'A';                 // Вывод значений        System.out.println("Имя: " + name);        System.out.println("Возраст: " + age + " лет");        System.out.println("Рост: " + height + " м");        System.out.println("Студент: " + isStudent);        System.out.println("Группа крови: " + bloodType);                 // Арифметические операции        int nextAge = age + 1;        double weight = 75.5;        double bmi = weight / (height * height);                 System.out.println("BMI: " + bmi);        System.out.println("Возраст через год: " + nextAge);    } }`

**Вывод программы:**

text

`Имя: Иван Возраст: 30 лет Рост: 1.75 м Студент: true Группа крови: A BMI: 24.48979591836735 Возраст через год: 31`

## Именование переменных

## Правила:

- Начинается с буквы, `_` или `$`
    
- Может содержать буквы, цифры, `_`, `$`
    
- **Не используйте** ключевые слова Java (`int`, `class`, `public` и т.д.)
    

## Конвенции (camelCase):

java

`// ✅ Хорошо int userAge; String firstName; double monthlySalary; boolean isLoggedIn; // ❌ Плохо int age1; String name_of_user; double SALARY;`

## Константы (final)

Переменная с модификатором `final` — это **константа**, её значение нельзя изменить.

java

`public class ConstantsExample {     public static void main(String[] args) {        final double PI = 3.14159;        final int MAX_USERS = 100;                 System.out.println("PI = " + PI);        System.out.println("Максимум пользователей: " + MAX_USERS);                 // PI = 3.14;  // ❌ Ошибка компиляции!    } }`

## Области видимости

java

`public class ScopeExample {     public static void main(String[] args) {        int x = 10;  // локальная переменная                 if (x > 5) {            int y = 20;  // видна только внутри if            System.out.println("x + y = " + (x + y));        }                 // System.out.println(y);  // ❌ Ошибка! y недоступна                 for (int i = 0; i < 3; i++) {            System.out.println("i = " + i);        }                 // System.out.println(i);  // ❌ Ошибка! i недоступна    } }`
