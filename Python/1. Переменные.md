# Урок 1: Переменные в Python

## Что такое переменная?

Переменная в Python — это **ссылка на объект** в памяти. Python **динамически типизирован** — тип определяется автоматически, имя переменной **присваивается** значению.

python

`переменная = значение  # вот и всё!`

## Объявление переменных

**Просто присваивание (без ключевых слов):**

python

`age = 25 salary = 75000.5 name = "Алексей" is_active = True grade = 'A'`

**Переменные создаются при первом присваивании!**

## Типы данных

## Примитивные типы

|Тип|Описание|Пример|
|---|---|---|
|`int`|Целое число (произвольный размер)|`42`, `1000000`|
|`float`|Дробное число|`3.14`, `2.0`|
|`str`|Строка|`"текст"`, `'текст'`|
|`bool`|Логическое (True/False)|`True`, `False`|
|`NoneType`|Отсутствие значения|`None`|

## Сложные типы

python

`person = {"name": "Иван", "age": 30}  # словарь colors = ["красный", "синий"]         # список numbers = (1, 2, 3)                   # кортеж (неизменяемый)`

## Полный пример программы

python

`# Объявление переменных age = 30 height = 1.75 name = "Иван" is_student = True blood_type = 'A' # Вывод значений print(f"Имя: {name}") print(f"Возраст: {age} лет") print(f"Рост: {height} м") print(f"Студент: {is_student}") print(f"Группа крови: {blood_type}") # Арифметические операции next_age = age + 1 weight = 75.5 bmi = weight / (height ** 2)  # ** — возведение в степень print(f"BMI: {bmi}") print(f"Возраст через год: {next_age}")`

**Вывод:**

text

`Имя: Иван Возраст: 30 лет Рост: 1.75 м Студент: True Группа крови: A BMI: 24.489795918367346 Возраст через год: 31`

## Именование переменных

## Правила:

- Начинается с буквы или `_`
    
- Может содержать буквы, цифры, `_`
    
- **Регистр важен!** `age` ≠ `Age`
    
- **snake_case** — стандарт Python
    

## Конвенции:

python

`# ✅ Хорошо (snake_case) user_age = 25 first_name = "Иван" monthly_salary = 75000 is_logged_in = True # ❌ Плохо age1 = 25 firstName = "Иван" MONTHLY_SALARY = 75000  # для констант OK`

## Области видимости

**Лексическая область видимости:**

python

`x = 10  # глобальная if x > 5:     y = 20  # видна после if!    print(f"x + y = {x + y}") print(x)    # ✅ 10 print(y)    # ✅ 20 (если была инициализация) for i in range(3):     print(f"i = {i}") # print(i)  # ❌ NameError после for`

## Проверка типов

python

`age = 25 name = "Иван" print(type(age))   # <class 'int'> print(type(name))  # <class 'str'> # Динамическое изменение типа age = "двадцать пять" print(type(age))   # <class 'str'>`

## Автоматическое приведение типов

python

`num = 5 text = "10" print(num + text)     # TypeError! print(num + int(text)) # 15 print(str(num) + text) # "510"`

## Множественное присваивание

python
